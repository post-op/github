# GitHub Workflow Repository

A collection of reusable GitHub Actions workflows used across internal repositories.

## Available Workflows

### Docker Build & Push
- `.github/workflows/docker_build.yml`
- [x] Builds and pushes Docker images to container registry
- [ ] Supports multi-arch builds
- [ ] Includes caching and security scanning

## Usage

1. Copy desired workflow file to your repository's `.github/workflows` directory
2. Configure required secrets and variables
3. Customize workflow as needed
